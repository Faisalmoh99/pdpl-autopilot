# pdpl-autopilot

An automated compliance monitoring system for Saudi Arabia's Personal Data Protection Law (PDPL).

> **Learning project** — built to practice real-world backend engineering while solving a genuine compliance problem.

## Planned Stack

| Layer | Technology |
|---|---|
| API | FastAPI (Python) |
| Database | PostgreSQL |
| AI / Analysis | Google Gemini |
| Containerisation | Docker |
| Hosting | Hetzner Cloud |

## Repository Structure

```
docs/        – architecture decisions (docs/adr/) and product definition
build-log/   – weekly progress notes
src/         – application source code
tests/       – automated tests
```
